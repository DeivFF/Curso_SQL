DDL, ou Linguagem de Definição de Dados (em inglês, Data Definition Language), é uma categoria de linguagem SQL utilizada para definir e gerenciar a estrutura de um banco de dados. Ela trata das operações relacionadas à criação, modificação e exclusão de objetos de banco de dados, como tabelas, índices, e visões. Vamos explorar algumas das principais operações e comandos DDL:

O comando CREATE é utilizado para criar objetos de banco de dados, como tabelas, índices, visões, e outros. Por exemplo:

CREATE TABLE tabela_exemplo (
    ID INT PRIMARY KEY,
    Nome VARCHAR(50),
    Idade INT
);

O comando ALTER é utilizado para modificar a estrutura de objetos existentes no banco de dados. Pode ser usado para adicionar, modificar ou excluir colunas de uma tabela, entre outras alterações.

-- Adicionar uma nova coluna
ALTER TABLE NomeDaTabela
ADD NovaColuna TIPO_DE_DADO;

-- Modificar o tipo de dados de uma coluna
ALTER TABLE NomeDaTabela
ALTER COLUMN NomeDaColuna TIPO_DE_DADO;

-- Excluir uma coluna
ALTER TABLE NomeDaTabela
DROP COLUMN ColunaParaExcluir;

O comando INDEX é utilizado em SQL para criar índices em tabelas, o que melhora o desempenho das consultas ao acelerar a recuperação de dados. Um índice é uma estrutura de dados que permite ao sistema de gerenciamento de banco de dados (SGBD) localizar e acessar rapidamente os registros em uma tabela com base nos valores das colunas incluídas no índice. Vamos explorar os principais aspectos relacionados ao comando INDEX:

A sintaxe básica para criar um índice é a seguinte:

CREATE INDEX NomeDoIndice
ON NomeDaTabela (Coluna1, Coluna2, ...);

NomeDoIndice: O nome dado ao índice, que deve ser exclusivo na tabela.
NomeDaTabela: O nome da tabela na qual o índice será criado.
(Coluna1, Coluna2, ...): As colunas pelas quais o índice será construído.

Tipos de Índices Comuns:

Índice Simples: Criado em uma única coluna.

CREATE INDEX idx_nome
ON TabelaExemplo (Nome);

Índice Composto: Criado em várias colunas.

CREATE INDEX idx_nome_idade
ON TabelaExemplo (Nome, Idade);

Índice Único (UNIQUE): Garante que os valores nas colunas indexadas sejam únicos.

CREATE UNIQUE INDEX idx_matricula
ON Alunos (Matricula);
